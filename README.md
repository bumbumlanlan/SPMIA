# SPMIA - Speaking Performance & Multiple Intelligences Analysis

A data analysis project examining the relationship between Multiple Intelligences (MI) theory and English speaking performance among students.

## Overview

This project processes survey data that measures:
- **8 Intelligence Types**: Intrapersonal, Interpersonal, Logical-Mathematical, Bodily-Kinesthetic, Linguistic, Naturalistic, Visual-Spatial, and Musical
- **Perception of MI Application**: How students perceive MI-based learning in classroom and self-study contexts
- **Dominant Intelligence**: Identified strongest intelligence type per participant
